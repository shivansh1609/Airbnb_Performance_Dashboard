# 🏡 Airbnb Performance Dashboard

## 📊 Overview

This project presents an **interactive Power BI dashboard** that analyzes Airbnb listing performance across different dimensions such as pricing, availability, reviews, and host behavior.

The goal is to provide **data-driven insights** for better decision-making by identifying trends, patterns, and key performance indicators.

---

## 🚀 Key Features

* 📈 **Revenue & Pricing Analysis**
* ⭐ **Review & Rating Insights**
* 🧑‍💼 **Host Performance Tracking**
* 🏙️ **City-wise & Property-wise Breakdown**
* 📊 **Pareto Analysis (80/20 Rule)**
* 🔍 Interactive filters and drill-down capabilities

---

## 🧠 Insights Derived

* A small percentage of hosts contribute to the majority of reviews (Pareto principle)
* Pricing varies significantly based on location and property type
* Highly rated listings tend to have better availability and engagement
* Certain cities dominate in terms of booking activity
